Configuración necesaria

Para que funcione correctamente, es necesario agregar el usuario a los siguientes grupos:

  • Contabilidad Analítica
  • Mostrar Características de Contabilidad Completas

Después de esto, solo hay que reiniciar el navegador con CTRL + R

Nueva función: UEPS

Se ha agregado la opción de UEPS (Últimas Entradas, Primeras Salidas) como método de costo.
